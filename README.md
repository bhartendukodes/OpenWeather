# OpenWeather - Weather Forecast App

A modern Android weather application providing real-time weather data and forecasts using the OpenWeatherMap API.

## Features

- Current weather conditions
- Multi-day weather forecast
- Location-based weather data
- Search weather by city name
- Temperature, humidity, wind speed details
- Beautiful weather icons and UI

## Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **Networking:** Retrofit
- **API:** OpenWeatherMap
- **Location:** Google Play Services
- **UI:** Material Design

## Getting Started

1. Clone the repository
```
git clone https://github.com/bhartendukodes/OpenWeather.git
```
2. Open in Android Studio
3. Add your OpenWeatherMap API key in the config
4. Build and run
